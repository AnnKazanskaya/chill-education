# Chill Education — Chat & Chill

Лендинг онлайн-школы английского «Chat & Chill».
Чистый HTML/CSS + vanilla JS (переключатель валют, карусель отзывов, форма заявки).

## Локальный запуск
Открой `index.html` в браузере — или подними простой сервер:

```bash
python3 -m http.server 8000
```

Затем открой http://localhost:8000
